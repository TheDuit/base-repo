# About Base System

Base System e uma fundacao generica para novos sistemas web.

Ela entrega:

- autenticacao JWT;
- usuarios administrativos;
- dashboard web autenticado;
- API NestJS;
- Web Next.js;
- PostgreSQL com migrations;
- Docker local/producao;
- Terraform AWS.

Cada fork deve alterar `system.manifest.json` antes de iniciar o produto.
